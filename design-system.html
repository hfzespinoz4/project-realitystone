Typeface roles

Cormorant Garamond

Display-only. Hero headlines, onboarding welcome, Society Branch signage, card face brand mark. Weights 300 and 500 only. Never in body copy or UI components.

F8 Bank

Inter

All UI text. Designed for screen legibility at small sizes. Weights 400, 500, 600, 700. Tabular numerals for all financial data.

Regular 400 — body copy Medium 500 — labels SemiBold 600 — emphasis Bold 700 — figures, CTAs

Iconography

Library

Phosphor Icons — Regular weight (1.5px stroke). One library only; no mixing with other icon sets, no emoji in UI.

Sizes

16px — inline text, metadata
20px — list items, form icons
24px — navigation, primary actions
32px — feature highlights, empty states

Colour contexts

Living Green — success / approved
Mara Flame — warning / attention
Terracotta — error / declined
Warm Stone — neutral / pending

Motion

Reveal

.reveal — opacity 0 + translateY(18px) → visible on intersection. Duration 700ms ease. Applied to every card, section, and content block.

Durations

Micro (button press) 80ms ease-out
Fast (dropdown, tooltip) 150ms ease-in-out
Standard (screen, modal) 250ms ease-in-out
Reveal (scroll trigger) 700ms ease

Rules

  • Never animate a transaction amount — show new value instantly
  • No looping animations on financial data screens
  • Always respect prefers-reduced-motion
  • Scroll-progress bar: Flame → Living Green gradient

Layout & spacing

Grid

Mobile-first. 4-column mobile · 12-column desktop. Max content width 1200px. Outer padding: clamp(16px, 3vw, 24px).

Base unit

4px. All spacing is a multiple of 4: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80. Border radius: 16px cards, 12px list items, 999px pills and buttons.

Touch targets

Minimum 44×44px for all interactive elements. Financial action buttons minimum 48×48px.
